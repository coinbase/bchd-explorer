# BCHD Explorer

Simple block explorer for the BCHD gRPC API with SLP token support.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Runs tests
```
npm test
```
